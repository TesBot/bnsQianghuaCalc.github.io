# 剑灵怀旧服（仙幻赛季）装备强化计算器

一个纯静态H5工具，帮助玩家计算装备强化材料消耗和最优开箱策略。

<img width="1479" height="926" alt="image" src="https://github.com/user-attachments/assets/572297d7-7016-480d-9e8b-abd81a910834" />


## 功能特点

- **两种计算模式**
  - 计算升级目标消耗：输入目标等级，计算所需材料和金币
  - 计算可强化最大等级：根据已有材料，计算能达到的最高等级

- **智能资源规划**
  - 自动计算开箱方案
  - 支持优先分解/优先合成两种策略
  - 多品阶材料转换链优化

- **完整数据支持**
  - 7种装备类型（武器、戒指、耳环、项链、手镯、腰带、手套）
  - 4种品阶（普通、耀眼、觉醒、灿烂）
  - 红字属性加成计算

- **用户体验**
  - 移动端优先响应式设计
  - 数据本地持久化（localStorage）
  - 详细的分步转换说明

## 使用方法

直接用浏览器打开 `index.html` 即可使用，无需安装任何依赖。

```
bnsQianghuaCalc/
├── index.html          # 主页面
├── css/style.css       # 样式文件
├── js/
│   ├── data.js         # 游戏数据
│   ├── calculator.js   # 核心算法
│   └── app.js          # UI逻辑
├── images/             # 装备图片
└── BnS.ico             # 网站图标
```

## 技术栈

- 纯静态 HTML/CSS/JavaScript
- 无框架依赖
- 本地存储数据持久化
